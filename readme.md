# mUI 

 Framework for creating apps with GUI written in C++

### Dependencies

 - OpenGL
 - GLEW
 - GLFW
 - nanovg
   - stb_image
   - stb_truetype

### What's works

 - [ ] Create OpenGL context
 - [ ] Application context
 - [ ] Widgets
       - [ ] Text-based widgets
       - [ ] Image-based widgets
 - [ ] Layouts

 Widget and Layout names taken from Android OS
